Creacion de un Nuevo Usuario Usuario a Registrar

Nombre.- Ingresar el nombre del nuevo Usuario.

Apellido.- Ingresar el apellido del nuevo Usuario.

Cedula.- Ingresar el documento de identificacion que consta de 10 digitos.

Correo.- Ingresar un correo al cual se pueda contactar con el usuario.

Direccion.- Ingresar la ciudad o calle donde se encuentra localizado el Usuario.

Telefono.- Ingresar algun tipo de numero telefonico con el cual se pueda contactar con el Usuario.
